# Video Export

Use render-video.js for HTML animation capture. Confirm dimensions, fps, duration, first frame, end frame, font loading, and optional audio mix. Use GIF optimization when delivering GIFs.

## Practical Rules

- Start from the user's intended audience and deliverable.
- Prefer concrete content over placeholder decoration.
- Use the starter assets in this skill when they save time or improve consistency.
- Verify visually before final delivery.
- Keep output in English unless the user asks for another language.

## Quality Bar

The result should look intentional, inspectable, and useful. Avoid generic gradients, empty marketing sections, vague copy, and ornamental motion that does not clarify the design.
