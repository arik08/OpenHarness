# Showcase Index

This folder contains static HTML and PNG examples used as visual references for Huashu Design. Use them to understand composition families and interaction tone, not as templates to copy blindly.

## Families

- Cover: strong first-frame composition and launch-style hierarchy.
- Infographic: editorial layout, dense information, and print-aware rhythm.
- PPT: presentation pacing and slide-scale typography.
- AI navigation, AI writing, developer docs, homepage, and SaaS: website-oriented examples with different product contexts.

When using these showcases, choose the direction that fits the user's audience and deliverable. Keep final copy in English unless instructed otherwise.
