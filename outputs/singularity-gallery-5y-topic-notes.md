# 특이점 갤러리 5년 주제 변화 조사 메모

작성일: 2026-05-04  
대상: DCInside `특이점이 온다` 마이너 갤러리(`thesingularity`) 공개 접근 가능 표본

## 수집 방식

- DCInside 갤러리 목록 검색을 직접 조회했습니다.
  - 키워드: `GPT`, `AGI`, `ASI`, `ChatGPT`, `Claude`, `Gemini`, `Sora`, `스테이블 디퓨전`, `로봇`, `일자리`, `오픈소스`, `중국`, `안전`, `규제`, `딥시크`, `이미지`, `영상`, `특이점`
  - 각 키워드별 1, 10, 30, 60페이지를 일부 확인했습니다.
- 검색엔진에 색인된 DCInside 게시글 스니펫을 보조로 확인했습니다.
- 전체 게시글/댓글 전수조사가 아니라 공개 접근 가능한 표본 기반 해석입니다.
- 작성자 닉네임, IP, 댓글 작성자 등 개인 식별 가능 정보는 보고서에 싣지 않았습니다.

## 해석 요약

1. 2021~2022년에는 기존 특이점/AGI 기대 담론 위에 이미지 생성 AI와 ChatGPT가 들어오며 체험형 논의가 생겼습니다.
2. 2023년은 GPT-4가 중심축이 된 해입니다. AGI 타임라인, 멀티모달, DALL-E 3, LLM 이해/한계 논의가 급격히 늘었습니다.
3. 2024년에는 GPT 단일 축보다 Claude/Gemini/o1/Sora 등 모델·모달리티·추론 방식 비교가 강해졌습니다.
4. 2025년에는 GPT-5, Gemini, Claude, DeepSeek/중국 모델, 오픈소스, 기업 갈등, 일자리와 규제까지 담론이 산업·사회 레이어로 확장되었습니다.
5. 2026년 1~5월 표본에서는 고성능 모델을 당연한 도구로 쓰는 분위기와 함께, 업무 자동화·코딩·이미지 기능·구독/서비스 품질 같은 실사용 이슈가 강합니다.

## 한계

- DCInside 검색은 접근 제한, 검색 정렬, 삭제글/비공개/이미지 본문 의존성 때문에 전수 빈도 분석에 적합하지 않습니다.
- 보고서의 차트 점수는 정밀 통계가 아니라 표본 기반 상대 강도입니다.
- 검색엔진 스니펫은 게시글 본문 전체가 아니므로 문맥 일부가 누락될 수 있습니다.
- 갤러리 여론은 활동 사용자와 특정 이벤트 당일의 게시량에 민감합니다.
