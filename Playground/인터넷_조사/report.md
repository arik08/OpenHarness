# HTML 미니 게임 보고서

## 1. 개요

이 프로젝트는 HTML, CSS, JavaScript만으로 만든 간단한 웹 게임입니다.  
플레이어는 화면 안에서 캐릭터를 움직이며 별을 먹고, 움직이는 폭탄을 피해야 합니다.

## 2. 게임 제목

**별 먹고 폭탄 피하기**

## 3. 사용 기술

- HTML
- CSS
- JavaScript
- Canvas API
- LocalStorage

## 4. 게임 방법

1. `index.html` 파일을 브라우저에서 실행합니다.
2. 방향키 또는 WASD 키로 초록색 플레이어를 이동합니다.
3. 화면에 나타나는 별을 먹으면 점수가 올라갑니다.
4. 빨간 폭탄에 닿으면 게임이 종료됩니다.
5. 게임 오버 후 스페이스바를 누르면 다시 시작할 수 있습니다.

## 5. 주요 기능

- 플레이어 이동 기능
- 별 아이템 생성 및 획득
- 폭탄 이동 및 충돌 판정
- 점수 표시
- 최고점 저장
- 게임 오버 및 재시작 기능

## 6. 파일 구성

| 파일명 | 설명 |
|---|---|
| `index.html` | 게임 실행 파일 |
| `report.md` | 게임 보고서 파일 |

## 7. 결론

이 게임은 별을 먹고 폭탄을 피하는 간단한 구조의 HTML 게임입니다.  
복잡한 설치 없이 웹 브라우저만 있으면 실행할 수 있으며, JavaScript의 기본적인 게임 로직과 Canvas 사용법을 연습하기에 적합합니다.
